# Smart Budget — Умный трекер расходов и доходов

**Smart Budget** — это современное веб-приложение на React для детального учета личных финансов. Приложение помогает контролировать бюджет, анализировать траты по категориям и следить за финансовой динамикой с помощью наглядных графиков.

Live demo: https://smart-budget-app-ten.vercel.app/

## Ключевые возможности

* **Мультивалютность:** Добавление записей в разных валютах (KGS, USD, EUR, RUB) с автоматической конвертацией в базовую валюту для подсчета общего баланса.
* **Лимиты расходов:** Установка ежемесячного бюджета с визуальным прогресс-баром и системой предупреждений при превышении лимита.
* **Продвинутая аналитика:** * Столбчатые диаграммы для анализа трат по категориям.
  * Линейные графики для отслеживания ежемесячной динамики расходов.
* **Умный поиск и фильтрация:** Поиск по описанию, фильтрация по типу (доход/расход) и конкретным категориям.
* **Сортировка данных:** По дате, сумме и по алфавиту.
* **Полное управление записями (CRUD):** Создание, просмотр, редактирование и удаление транзакций.
* **Визуализация категорий:** Уникальные эмодзи-иконки и цветовое кодирование для каждой категории расходов и доходов.
* **Локальное хранение:** Все данные и настройки (включая лимит бюджета) надежно сохраняются в памяти браузера (`localStorage`).

## Технологии и инструменты

* **React (Hooks, Context API, useReducer):** Управление сложным глобальным состоянием приложения.
* **JavaScript (ES6+):** Логика конвертации валют, фильтрации и группировки массивов данных.
* **CSS3:** Адаптивный дизайн, CSS-переменные, Flexbox.
* **Recharts:** Построение интерактивных SVG-графиков для аналитики.

Разработано в качестве пет-проекта для демонстрации навыков работы с React и сложной логикой управления данными.
